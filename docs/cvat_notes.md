## Экспорт и импорт COCO в CVAT

- Экспорт из CVAT: обычно выбираешь COCO 1.0, получаешь JSON + картинки.
- Этот проект читает COCO JSON и учит модели.
- Предсказания сохраняются в формате COCO predictions (bbox + category_id + score).

Дальше варианты:

2) Для полуавторазметки можно писать конвертер в формат CVAT или использовать CVAT server API.
